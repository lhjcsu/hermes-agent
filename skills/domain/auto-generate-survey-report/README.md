# 自动生成工程勘察报告

从华宁勘察软件导出的 Excel 数据自动生成岩土工程勘察报告（.docx）。

## 功能

- 读取华宁导出的 .XLS / .xlsx 数据文件
- 以正式勘察报告 .docx 为模板
- 自动填充 25+ 张数据表格（标贯、动探、土工试验、物理力学指标、承载力等）
- 更新数据相关段落（工作量、地层描述等）
- 一键输出完整报告

## 依赖

```bash
pip install xlrd openpyxl python-docx pywin32 -i https://pypi.tuna.tsinghua.edu.cn/simple
```

## 用法

```bash
python generate_report.py
```

## 工作流程

```
华宁导出 Excel (.XLS/.xlsx) 
        ↓
Python 读取（xlrd/openpyxl）
        ↓
加载正式报告 .docx 模板
        ↓
python-docx 填充表格 + 更新段落
        ↓
输出完整报告 .docx
```

## 注意事项

- `.XLS` 老格式必须用 `xlrd`
- `.doc` 转 `.docx` 需要 `win32com`（需安装 Office）
- 数据量大的 Excel 用 `read_only=True` 提升性能
